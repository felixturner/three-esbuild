# three-esbuild

Minimal boilerplate using Three.js with esbuild.

Using Three.js, ES6 imports, Orbit controls, Stats.js.

## Build Dev

- `npm install`
- `npm run dev`
- Navigate to `http://localhost:8000/`

Reload browser to rebuild JS.

## Build Production

- `npm run build`
- Production files are created in `dist` folder
